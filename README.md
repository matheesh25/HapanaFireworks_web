# 🎆 Hapana Fireworks Web System


A complete **E-commerce and Management System** developed for Hapana Fireworks.  
This system provides a modern UI with full backend integration using **PHP & MySQL**, designed to manage products, users, orders, suppliers, and delivery operations efficiently.

---

## 🚀 Project Overview

Hapana Fireworks Web System is a full-featured web application that allows customers to browse and purchase fireworks while providing administrators with powerful management tools.

The system includes:
- User-friendly customer interface
- Advanced admin dashboard
- Secure database integration
- Automated report generation
- Email notification system
